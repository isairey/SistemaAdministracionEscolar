# 🎓 Sistema de Administración Escolar

Sistema web desarrollado para la gestión eficiente de instituciones educativas, permitiendo administrar alumnos, docentes, materias y procesos escolares de forma sencilla e intuitiva.

---

## 📌 Descripción

El **Sistema de Administración Escolar** es una aplicación web que facilita la organización de información académica. Está diseñado con tecnologías básicas del lado del cliente, lo que lo hace ligero, rápido y fácil de desplegar.

Este sistema permite llevar un control estructurado de los datos escolares sin necesidad de configuraciones complejas o servidores avanzados.

---

## 🚀 Características

- 📋 Registro de alumnos
- 👨‍🏫 Gestión de docentes
- 📚 Administración de materias
- 🗂️ Organización de grupos
- 🔍 Búsqueda de registros
- 🧾 Visualización de información en tablas dinámicas
- 💾 Almacenamiento local (LocalStorage)

---

## 🛠️ Tecnologías utilizadas

- **HTML5** → Estructura de la aplicación  
- **CSS3** → Estilos y diseño responsivo  
- **JavaScript (Vanilla)** → Lógica y funcionalidad

## ⚙️ Instalación y uso

1. Clona este repositorio:
```bash
git clone https://github.com/tuusuario/sistema-escolar.git
```
Accede a la carpeta del proyecto:
```
cd sistema-escolar
```
Abre el archivo index.html en tu navegador.
## 💡 Funcionamiento

El sistema funciona completamente en el navegador utilizando LocalStorage, lo que permite guardar la información sin necesidad de base de datos.

Los datos se almacenan de manera local, por lo que no se pierden al recargar la página.
